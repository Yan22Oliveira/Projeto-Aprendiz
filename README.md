# Paleta de cores

**Obs:** o dart não interpreta diretamente com #, para utilizar é preciso trocar o # para 0xFF

Para utilizar uma cor no dart basta chamar Color(_inserir_hexadecimal_sem_#)
```dart
Color(0xFFFF941A); //Exemplo de uso
```
